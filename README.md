# Redir_downloads
### Run redir_downloads to redirect
- ~/Downloads/{*.jpg, *.jpeg, *.png,*.gif}  ->  ~/Pictures/{*.jpg, *.jpeg, *.png,*.gif}
- ~/Downloads/{*.docx, *.doc} ->  ~/Documents/{*.docx, *.doc}
- ~/Downloads/{*.pdf} ->  ~/Pdfs/{*.pdf}
- ~/Downloads/{*.exe} ->  ~/Executables/{*.exe}
